    1. HTML5 简介

    HTML5 是最新的 HTML 标准，提供了新的语义元素、图形和多媒体支持、以及更好的 API。

    2. 新的语义元素

    • <header> - 页眉
    • <footer> - 页脚
    • <article> - 文章
    • <section> - 区块
    • <nav> - 导航
    • <aside> - 侧边栏

    2.1 新的结构元素

    
<body>
    <header>
        <nav>导航菜单</nav>
    </header>
    
    <main>
        <article>
            <section>第一部分</section>
            <section>第二部分</section>
        </article>
        
        <aside>侧边栏</aside>
    </main>
    
    <footer>页脚</footer>
</body>

    3. 图形和多媒体

    
<canvas id="myCanvas" width="200" height="100"></canvas>


<video controls>
  <source src="movie.mp4" type="video/mp4">
</video>

    4. 表单增强

    • <input type="email"> - 邮箱输入
    • <input type="date"> - 日期选择
    • <input type="range"> - 范围选择
    • <input type="color"> - 颜色选择

    5. HTML5 API

    主要 API：
    • 地理定位 API (Geolocation)
    • 拖放 API (Drag and Drop)
    • Web Storage
    • Web Workers
    • WebSocket
    // 地理定位示例
if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(
        position => {
            const latitude = position.coords.latitude;
            const longitude = position.coords.longitude;
            console.log(`位置：${latitude}, ${longitude}`);
        },
        error => {
            console.error('获取位置失败：', error);
        }
    );
}
