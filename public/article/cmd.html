CMD命令大全

Windows命令提示符(CMD)常用命令详解

目录

    1. 基础命令

    dir

    dir [路径] [/参数]

    显示目录中的文件和子目录列表

    • /w - 宽格式显示
    • /p - 分页显示
    • /a - 显示具有指定属性的文件
    • /s - 显示指定目录和所有子目录中的文件
    • /b - 使用空格格式(没有标题信息或摘要)

    cd (Change Directory)

    cd [路径]

    更改当前目录或显示当前目录名

    • cd .. - 返回上级目录
    • cd \ - 返回根目录
    • cd /d D: - 切换到其他盘符
    • cd. - 显示当前目录路径

    2. 文件操作

    copy

    copy [源文件] [目标路径]

    复制文件到另一个位置

    del

    del [文件名]

    删除一个或多个文件

    move

    move [源文件] [目标路径]

    移动文件到新位置

    3. 目录操作

    mkdir

    mkdir [目录名]

    创建新目录

    rmdir

    rmdir [目录名] [/s] [/q]

    删除目录

    • /s - 删除所有子目录和文件
    • /q - 安静模式

    4. 网络命令

    ipconfig

    ipconfig [/all] [/release] [/renew]

    显示和管理网络配置

    • /all - 显示完整配置信息
    • /release - 释放指定适配器的IP地址
    • /renew - 更新指定适配器的IP地址
    • /flushdns - 清除DNS解析程序缓存

    ping

    ping [主机名或IP]

    测试网络连接

    netstat

    netstat [-an]

    显示协议统计和当前TCP/IP网络连接

    5. 系统命令

    tasklist

    tasklist [/参数]

    显示当前运行的进程列表

    taskkill

    taskkill /pid [进程ID] 或 /im [进程名]

    终止进程

    systeminfo

    systeminfo

    显示系统信息

    6. 高级命令

    sfc

    sfc /scannow

    系统文件检查器

    chkdsk

    chkdsk [盘符:] [/f] [/r]

    检查磁盘并显示状态报告

    7. 使用技巧

    • 使用上下方向键可以查看命令历史
    • 使用Tab键可以自动补全文件名和目录名
    • 使用cls命令可以清空屏幕
    • 使用help命令可以查看所有可用命令
    • 在命令后加/?可以查看该命令的帮助信息
